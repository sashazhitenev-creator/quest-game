# КВЕСТ — браузерная текстовая AI-игра

Текстовое приключение в духе игры из Рика и Морти: нейросеть (Groq, Llama 3.3 70B) выступает гейм-мастером, игрок пишет любые действия свободным текстом, игра генерирует исходы и ведёт к финалу.

## Архитектура

```
Браузер игрока → Netlify Function (chat.js) → Groq API
```

Ключ Groq хранится только на сервере (переменная окружения), в браузер он не попадает. Системный промпт тоже живёт на сервере — игроки не могут его подменить. Сервер Netlify находится за границей, поэтому VPN для доступа к Groq не нужен, а игроки из России открывают сайт напрямую.

## Деплой (5 минут)

1. **Ключ Groq.** Зайди на https://console.groq.com → API Keys → Create API Key. Скопируй ключ.

2. **Репозиторий.** Залей эту папку в новый репозиторий на GitHub (можно прямо через веб-интерфейс: New repository → Add file → Upload files).

3. **Netlify.** На https://app.netlify.com → Add new site → Import an existing project → выбери репозиторий. Настройки сборки не трогай (netlify.toml всё описывает). Deploy.

4. **Ключ в окружение.** Site configuration → Environment variables → Add a variable:
   - Key: `GROQ_API_KEY`
   - Value: твой ключ из шага 1
   
   После добавления сделай передеплой: Deploys → Trigger deploy → Deploy site.

5. Готово. Игра доступна по адресу `https://твой-сайт.netlify.app`, можно привязать свой домен.

## Локальный запуск (для разработки)

```bash
npm install -g netlify-cli
export GROQ_API_KEY=твой_ключ   # Windows: set GROQ_API_KEY=...
netlify dev
```

Откроется http://localhost:8888 — функция будет работать локально. (Из России для локального запуска нужен VPN, т.к. запрос к Groq пойдёт с твоей машины. На проде это не нужно.)

## Настройка

- **Модель:** в `netlify/functions/chat.js` константа `MODEL`. Сейчас `llama-3.3-70b-versatile` — лучший русский на Groq. Для экономии лимитов можно `llama-3.1-8b-instant`, но качество сюжета просядет.
- **Промпт гейм-мастера:** там же, `SYSTEM_PROMPT`. Можно менять жанры, добавлять инвентарь, золото и т.п. — главное сохранять формат строк `STATUS:` и `FINAL:`, их парсит фронтенд.
- **Жанры:** массив `GENRES` в `index.html`.

## Лимиты Groq (бесплатный тариф)

У бесплатного тарифа есть лимиты запросов в минуту и в день на модель. Для небольшой аудитории хватает; если игра станет популярной, варианты: перейти на платный тариф Groq, кэшировать, или добавить простое ограничение частоты запросов в функции.

## Идеи для развития

- Инвентарь и золото через тот же трюк с маркерами (`STATUS: HP=80 | GOLD=15 | лес`)
- Кнопка «поделиться» финалом (текст эпитафии + число ходов)
- Таблица рекордов (кто дольше выжил) — Netlify Blobs или внешняя БД
- Портирование в Telegram-бота на aiogram: та же функция chat, история в SQLite по user_id
